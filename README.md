# Accelerated Forgetting (accforg)

## Project Overview

This repository contains data and analysis code for the **Accelerated Forgetting** project.

The project investigates mechanisms underlying accelerated memory decay, with a focus on the cognitive and temporal dynamics of forgetting. The purpose of this repository is to ensure transparency, reproducibility, and adherence to open science practices.

The repository includes:

- Data used for statistical analyses  
- R scripts for preprocessing and modeling  
- Output files and figures (if applicable)

## Analysis

All analyses were conducted in **R**.

To reproduce the analyses:

1. Open `alf_preliminary.R` in R or RStudio  
2. Set the working directory to the project root folder (`accforg/`)  
3. Ensure required packages are installed  
4. Run the script from top to bottom
